# Abhishek Pokar Electrical Services

A modern, professional, mobile-first electrician business website built with React and Tailwind CSS.

## Features
- Clean, premium UI
- Electric Blue, Dark Navy, White color palette
- Glassmorphism cards, subtle shadows
- Sticky header, floating WhatsApp & Call Now buttons
- Responsive, fast loading, SEO optimized
- Service cards with icons (Lucide)
- Projects gallery with lightbox modal
- Testimonials, pricing, contact form with success toast
- Scroll-to-top button, smooth scroll navigation
- Accessibility and alt tags

## Sections
- Hero
- About
- Services
- Why Choose Us
- Projects Gallery
- Testimonials
- Pricing
- Contact
- Footer

## Deployment
Ready for Vercel. To deploy:
1. Push this repo to GitHub.
2. Connect to Vercel and import the repo.
3. Vercel auto-detects Vite/React and deploys.

## Local Development
```bash
npm install
npm run dev
```

## Customization
- Replace image placeholders with real project photos.
- Update contact info and social links as needed.
